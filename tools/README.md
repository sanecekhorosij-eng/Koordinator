# Инструменты TECTOV

В этой директории будут располагаться конфигурации и документация для подключаемых инструментов.

## Планируемые инструменты

- **GitHub** — работа с репозиториями, issues, PR
- **Браузер** — автоматизация через Puppeteer/Playwright
- **Файловая система** — чтение/запись файлов
- **Другие API** — внешние сервисы

Инструменты регистрируются через `backend/agent/tools.js`.
